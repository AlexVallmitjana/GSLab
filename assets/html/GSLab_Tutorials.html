Tutorial 1 – Basic phasor analysis using lifetime measurement of a human skin biopsy

    File tab – Load files into the software

  1. Start GSLab.
  2. Right-click in the empty space and select Load > SimFCS ref.
  3. Select the image file from the Fig2_ABCD folder.
  4. The phasor plot and image will appear.
  5. Check/uncheck the file in the main panel to include/exclude it from the phasor plot.
  6. Hover over the phasor to see highlighted image regions.
  7. Use the scroll wheel to adjust the cursor size.
  8. Hover over the image to see the black cursor move on the phasor plot.

    9. Image tab – Control the rendering of the intensity image

  9. This image has a very high dynamic range, we can tune a better rendering: Observe the intensity histogram how all the values are in the far left.
  10. Move the Right Intensity Slider halfway to reveal collagen structures.
  11. Move it further (to about 1/8th) to see the epidermis, though some bright pixels may saturate.
  12. Checking/unchecking Excess to zero to exclude bright saturated pixels from the phasor plot.
  13. The image contains many low-photon pixels corresponding to outside the biopsy. We can threshold these pixels out:
  14. Uncheck Relative to show the histogram in real counts.
  15. Set the lower intensity threshold to 30 and press Enter.
  16. The phasor now excludes the pixels with low photon counts: the noisy distribution on the left part of the phasor plot has disappeared.

    17. Phasor tab – Select the phasor magnitude to measure

  17. In the bottom section try the different magnitudes and see how the color projection changes. Use TauPhase for this dataset. Each pixel in the image is colored according to the underlying colormap in the phasor.

    18. Cursors – Place cursors on the phasor plot

  18. Depending on the settings in the previous sections, the phasor distribution may appear very differently. We may see 1, 2 3 4 or more distinct populations.
  19. Click on the phasor plot to place cursors and color the image accordingly.
  20. Use the red X button (top-right) to remove cursors (appears if one or more cursors are on the phasor plot).

    21. Export tab – Save images and phasor plots

  21. Click Output Folder to choose a save location.
  22. Uncheck all checkboxes except for Phasor with axes, Phasor Plot as, Intensity image, Magnitude-colored x intensity, and Combine into single image.
  23. Click Export LR to save the selected items.
  24. A single file will be exported with grayscale intensity, phasor color projection, and the phasor plot.
  25. Uncheck Magnitude-colored x intensity and check the Cursor-Colored image.
  26. Press the Export LR button to export like before but now the image is colored according to any cursors on the phasor plot.

Tutorial 2 – Phasor calibration and phasor filtering

    File tab – Load files into the software

  1. Start GSLab.
  2. Right-click in the empty space and select Load > SimFCS ref.
  3. Select the file from the Fig2_ABCD folder.
  4. Right-click in the empty space and select Load > Trace txt/csv.
  5. Select the Coumarin6 file from the Fig2_ABCD folder (used for calibration).
  6. This file contains a single trace, the software displays it as a single-pixel image in the Image Window.
  7. Check/uncheck each file in the list to see their contributions in the phasor plot.
  8. Hover over the phasor plot or image window to locate the Coumarin6 on the phasor plot.
  9. Coumarin6 has a known lifetime of 2.5ns, its phasor position should be on the Universal Circle at the corresponding lifetime. We use this fact to calibrate the phasor coordinates of the biopsy values.
  10. Click on the Coumarin6 file name to select it in blue.
  11. Right-click and select Calibration file.
  12. The Calibration panel unfolds showing the selected file name in the list.
  13. Use the arrow button to expand/collapse the panel.
  14. Enter 2.5 in the Calibration Lifetime (ns) field. Press Enter.
  15. Check the Apply checkbox to shift the phasor plot accordingly.
  16. Change the Calibration Lifetime (ns) to see how it affects the plot.
  17. Once set to 2.5ns, we can remove the calibration file:
  18. Uncheck the Coumarin6 filename from the list.
  19. Right-click in the empty space and select Clear > Unchecked.

    20. Image tab – Control the rendering of the intensity image

  20. Slide the right slider down to about 1/8th of the range to adjust the image display.

    21. Phasor tab – Filter the phasor data

  21. Set Phasor filtering to 1 and press Enter (phasor plot will look noisy).
  22. Set it to 5 and press Enter (reduces noise).
  23. Set Times to 3 and press Enter (further sharpens the phasor).
  24. Higher filtering smooths out regions in the image.
  25. In Measured magnitude, click TauNorm to display the color projection to the Universal Circle.
  26. Adjust Color range to 0 and 0.5 to cover the full semicircle.
  27. Click Auto to let the software adjust the color range automatically.

    28. Display tab – Customize the phasor plot

  28. Select a different colormap from Magnitude colormap dropdown.
  29. Set Harmonic # to 2 (phasor computed in second harmonic).
  30. Change it back to 1 to return to the first harmonic.
  31. Toggle Lines and Ticks on/off.
  32. Adjust Font sizes.
  33. Toggle Soft Masks to see intensity-based or color-only images.
  34. Choose a different colormap from Density colormap.

    35. Export tab – Save images and phasor plots

  35. Click Output Folder to select a save location.
  36. Uncheck all checkboxes except for Print colorbar, and Magnitude-colored x intensity.
  37. Click Export LR to save the selected items.
  38. Two image files will be exported the image with phasor lifetime projection, and colorbar with the lifetime values.

Tutorial 3 – Automated phasor clustering

    File tab – Load files into the software

  1. Right-click in the empty white space and select Load > SimFCSref.
  2. Navigate to the folder location for the downloaded Fig2_ABCD and select the single file in the folder.

    3. Image tab – Set the image dynamic range

  3. Uncheck the Relative checkbox to display the histogram in real counts instead of normalized intensity
  4. In the Intensity threshold values, type 30 for the lower value field (press enter).
  5. Slide the right slider down to about 1/8th of the range to reveal the structure in the epidermis at the cost of saturating the brighter pixels.

    6. Cluster tab – Perform automated clustering on the phasor plot

  6. GMM attempts to find the linear combination of K gaussian distributions that best fits the data.
  7. Clicking on the plot to leave some cursors as starting points for the algorithm. The positions and sizes are not as important, the algorithm will tune them, but the total number of clusters will be set by the number of clusters.
  8. Click on Train GMM button to generate the model.
  9. The image will be colored according to the model results.
  10. Contour lines will appear on the phasor plot, representing the Gaussian components.
  11. Click on Contour separate clusters to observe the individual components on the phasor plot.
  12. Click on Ellipse with % of data to display ellipses instead
  13. Change the percentage to display bigger or smaller ellipses.
  14. Click Apply GMM to reapply the model after changes to the image or phasor settings.
  15. The Clear GMM button will clear the current model.
  16. With Probability assign to color each pixel based on its mix of components.
  17. Click Hard assign to color each pixel based on the highest component coefficient.
  18. Check Blending to smooth color transitions.
  19. Check Override magnitude for export to save GMM-clustered images instead of cursor-colored ones.

    20. Export tab – Save the results

  20. Click Output Folder to choose where to save the data.
  21. Two checkboxes now include the GMM-colored option.
  22. Uncheck all checkboxes except for Phasor with axes, Phasor Plot as, Intensity image, GMM-colored image, and Combine into single image.
  23. Click Export LR to save.
  24. One file will be exported, containing the grayscale intensity image, GMM-colored image, and phasor plot.
  25. Uncheck Combine into single image.
  26. Click Export LR again.
  27. Now, three separate images should appear in the output folder: phasor plot, grayscale intensity, and GMM-colored image.

Tutorial 4 – Phasor unmixing with nuclear dyes

    File tab – Load files into the software

  1. Start GSLab.
  2. Right-click in the empty space and select Load > SimFCS ref.
  3. Select the 5 files from the Fig2_EFGH folder. These are cell nuclei, 4 files have a single dye, the 5th contains all dyes.
  4. Check/uncheck files in the list to see their contributions in the phasor plot.
  5. The files can be rearranged for clarity: click on the file that contains the term ‘4dyes’ in the name to select it.
  6. Right-click and select Move > Bot.
  7. Click on the file that contains the term ‘EtBr’ in the name to select it.
  8. Right-click and select Move > Top.
  9. We should currently have the files sorted in descending lifetime order with the mixture file in the bottom.
  10. Check/uncheck files in the list to see their contributions in the phasor plot.

    11. Image tab – Control the rendering of the intensity image

  11. Slide the left slider up to about 1/10th of the range to exclude the pixels outside of the nuclei.
  12. Slide the right slider down to about 1/3rd of the range to adjust the image display.

    13. Phasor tab – Filter the phasor data

  13. Change the Phasor filtering value to 7. Press Enter.
  14. Change the Times value to 2. Press Enter.
  15. In the Measured magnitude section click on TauNorm.

    16. Display tab – Customize the phasor plot

  16. Check the Soft Masks.
  17. Check/uncheck Normalize by file to observe the contribution of the narrower distributions

    18. Unmix tab – Perform unmixing of 4 components

  18. Click the + button 4 times to add 4 pure components on the universal circle.
  19. Enter the values 0.5ns, 2ns, 5ns and 20ns respectively.
  20. The quadrilateral delimits the pure component locations.
  21. Check the options Export fraction im, Color components, Single image, and Export fraction x int.
  22. Click Compute LR to launch the unmixing and save the files (in the output folder set in the Export tab).
  23. The output is an image for each component with the color-coded unmixed photon fraction in each pixel.
  24. Check Group by File.
  25. Click Compute LR to launch the unmixing.
  26. The output is an image for each file with the color-coded unmixed photon fraction in each pixel.
  27. Check Export fraction csv.
  28. Click Compute LR to launch the unmixing and export a csv file for each image with the unmixed photon fraction at each individual pixel.

Tutorial 5 – Spectral phasor unmixing with spectrally resolved organelle dyes

    File tab – Load files into the software

  1. Start GSLab.
  2. Right-click in the empty space and select Load > Zeiss lsm.
  3. Select the file from the SpectralExample folder. This is a culture of cells loaded with 5 organelle dyes.
  4. Hover over the phasor to see highlighted image regions:
  5. Top right is the nuclear dye.
  6. Bottom right is the membrane dye.
  7. Bottom-left is the mitochondria dye.
  8. Middle-left is the Golgi dye.

    9. Image tab – Control the rendering of the intensity image

  9. Slide the right slider down to about 1/2 of the range to adjust the image display.

    10. Phasor tab – Filter the phasor data

  10. Change the Phasor filtering value to 3. Press Enter.
  11. Change the Times value to 1. Press Enter.
  12. In the Measured magnitude section click on Phase.
  13. Set the Color range to 0 and 1.

    14. Display tab – Customize the phasor plot

  14. Check the Soft Masks.
  15. Set the Spectral band range to 450 and 700nm respectively (matching the acquisition spectral detector).

    16. File tab – Load theoretical spectra downloaded from Chroma

  16. Right-click in the empty space and select Load > Trace txt/csv.
  17. Select the file from the SpectralExample folder. This is a list of the 5 emission spectra of the dyes.
  18. Hover over the phasor to find the five emission spectra as small circles.

    19. Unmix tab – Perform unmixing of 5 components

  19. In the Source radio buttons select File.
  20. The box on the left lists all files loaded. Check the 5 that correspond to the theoretical spectra of the dyes.
  21. An irregular pentagon is formed delimiting the pure component locations.
  22. Check the options Export fraction im, Color components, Single image, Export fraction x int, and Merge comps.
  23. Click Compute LR to launch the unmixing and save the files (in the output folder set in the Export tab).
  24. The output is an image for each component with the color-coded unmixed photon fraction in each pixel.
  25. The component colors can be changed with the cursor colors in the Display tab.
  26. We can go to the File tab, uncheck the pure component traces, go back to the Unmix tab and run Compute LR
  27. This will create the unmixing data for just the selected file.
